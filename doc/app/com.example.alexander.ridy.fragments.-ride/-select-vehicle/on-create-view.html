onCreateView

fun onCreateView(inflater: LayoutInflater, container: ViewGroup?, savedInstanceState: Bundle?): View?

onCreateView is called to "inflate" the layout of the fragment, i.e. graphic initialization is usually done here. It is always called after the onCreate method.

Parameters

inflater - the attr that handles the inflation of the fragment

container - attr that forms a container for the data to be displayed

savedInstanceState - the savedInstanceState bundle can be used to reload the previous state information.

Return
function returns a view, this will be displayed on the screen
